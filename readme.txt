Desenvolvendo o primeiro laboratório da disciplina de Sistemas Microcontrolados, foi possível adaptar-se a interface de
desenvolvimento, usando o simulador e seus comandos. Além disso, desenvolvendo o "Exercício 3" na linguagem assembly, foi 
possível detectar atribuições de informações aos registradores. Tudo isso, usando os comandos MOV, MNV e algumas linhas de 
configurações.
Pode-se perceber também, a diferença nos valores registrados quando há mudanças entre os comandos MVN E MOV.
